Keyboard Proxy Chunk

Keyboard Proxy Chunk token: ``KeyboardProxy''
Defines a pointer to some keyboard device. The device pointed to must be one of the known keyboard devices.
Device Property token: ``device''
The keyboard device to which the proxy points. Currently, the proxy must point to a Keyboard or simKeyboardDigital chunk. Other chunk types can be added through the chunk description file (chunksDesc).

Keyboard Proxy chunk definition

chunk KeyboardProxy "Keyboard Proxy" "This proxy points to a keyboard device."
  Name String 1 "Name" "Unique name of an instance of this chunk type"
  device Chunk 1 "Device" vj_enumeration { "Keyboard" "simKeyboardDigital" } 
          "The keyboard device to hook the proxy to."
  end
