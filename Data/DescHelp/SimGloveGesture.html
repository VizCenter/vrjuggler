Glove Gesture Simulator Chunk

Sim Glove Gesture Chunk token: ``SimGloveGesture''
Defines a simulated glove gesture device. It uses a user-specified keyboard proxy to accomplish this simulation.
Keyboard Proxy Property token: ``keyboardProxy''
Sets the proxy pointing to the keyboard to use for the simulation. The proxy must point to a KeyboardProxy chunk. More chunk types can be added through the chunk description file (chunksDesc).
Keypresses The user can define a number of keypresses that are used to control the gesture simulator. Each keypress consists of two parts: a modifier and a key.

Modifier Keys
Lists the modifier keys that may be used together with normal keyboard keys when simulating the glove gesture device. The modifier keys must be chosen from a predefined set of keys that are treated as modifiers. The current modifier keys are:

  • NONE (no modifier)
  • SHIFT (the <SHIFT> key)
  • CTRL (the <CTRL> or <CONTROL> key)
  • ALT (the <ALT> or <META> key)

Keys
The list of keyboard keys, mouse movement directions and mouse buttons that can be used for simulating the glove gesture device. All standard keys (except the F1-F12 keys), modifier keys (listed above), cursor movement keys, mouse movement directions and mouse buttons (for a three-button mouse) are available for use. Each key can be assigned to a particular aspect of the simulation, and modifiers can be used to allow the same keys (say those on the number pad) to be used for different aspects of simulation.

Trained Filename Property token: ``trainedFilename''
Names the file containing trained gestures. This is a string value, and as the name of a file, it depends upon the underlying file system for its syntax. On UNIX, file names are case sensitive, and directories are separated by the `/' character. On Win32-based systems, file names are not case sensitive, and directories are separated by the `\' character.
Glove Position Property token: ``glovePosition''
A position proxy for the glove's position in space. The proxy must point to another PosProxy chunk. Other allowed types can be added through the chunk description file (chunksDesc).

Sim Glove Gesture chunk definition

chunk SimGloveGesture "Glove Gesture Simulator" "Simulate glove gesture device"
  Name String 1 "Name" "Unique name of an instance of this chunk type"
  keyboardProxy Chunk 1 "Keyboard Proxy" vj_enumeration { "KeyboardProxy" } 
            "The keyboard to use."
  keyPairs EmbeddedChunk -1 "Key Presses" vj_enumeration { "KeyModPair" } ""
  trainedFilename String 1 "Trained File Name" 
            "The name of the trained gesture file.  Used to get simulated values."
  glovePos Chunk 1 "Glove Position" vj_enumeration { "PosProxy" } 
            "The position proxy for the glove position."
  end
