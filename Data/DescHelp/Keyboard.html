Keyboard Input Window Chunk

Keyboard Input Window Chunk token: ``Keyboard''
The Keyboard Input Window opens an on-screen window which can receive keypress and mouse events. Simulator devices are ususally attached to a Keyboard Proxy pointing to a Keyboard Window.
Width Property token: ``width''
Window width.
Height Property token: ``height''
Window height.
Origin Property token: ``origin''
The XY position of the window's lower-left corner on the screen. The upper-left corner of the screen is (0,0).
Mouse Sensitivity Property token: ``msens''
Mouse movements when the window is active are translated into individual keypress events. This factor controls the sensitivity of this translation. For example, a value of 1 means that 1 keypress will be generated for every pixel the mouse moves in a particular direction. A value of .1 means that 1 keypress will be generated for every 10 pixels of movement.

Keyboard Input Window chunk definition

chunk Keyboard "Keyboard" "Configures a window that accepts keyboard & mouse input"
  Name String 1 "Name" "Unique name of an instance of this chunk type"
  width Int 1 "width" "window width"
  height Int 1 "height" "window height"
  origin Int 2 "origin" vj_valuelabels { "X" "Y" } "Window origin (upper left is 0,0)"
  msens Float 1 "msens" "Mouse sensitivity, 1 is 1 keypress per pixel moved, 0.1 is 1 key for 10 pixels"
  end
