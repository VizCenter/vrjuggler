Digital Proxy Chunk

Digital Proxy Chunk token: ``DigProxy''
Defines a pointer to some digital data device. The device pointed to must be one of the known digital devices.
Device Property token: ``device''
The digital device to which the proxy points. The proxy points to another chunk that must be in a predefined set of types. Those types are currently:
  • SimDigital
  • IBox
More chunk types can be added through the chunk description file (chunksDesc).
Unit Property token: ``unit''
Sets a unique number for each digital proxy for use when querying the device with the getData() method. The numbering should be non-decreasing starting from 0.

Digital Proxy chunk definition

chunk DigProxy "DigProxy" ""
   Name String 1 "Name" "Unique name of an instance of this chunk type"
   device Chunk 1 "device" vj_enumeration { "SimDigital" "IBox" } "The device name to hook up the proxy to"
   unit Int 1 "unit" "The unit number in the device to query for getData's"
   end
