Input Manager Chunk

Input Manager Chunk token: ``InputManager''
Provides initial settings for the Input Manager.
Dummy Position Property token: ``dummyPos''
Sets the position for the dummy value used by the Input Manager. This is a triple of three floating-point values.
Dummy Orientation Property token: ``dummyOr''
Sets the Euler orientation for the dummy value used by the Input Manager. This is a triple of three floating-point values providing the x, y and z angles of rotation.

Input Manager chunk definition

chunk InputManager "InputManager" "Configure the initial Input Manager settings."
  Name String 1 "Name" "Unique name of an instance of this chunk type"
  dummyPos Float 3 "Dummy Position" vj_valuelabels { "X" "Y" "Z" } "The Position of the dummy value."
  dummyOr Float 3 "Dummy Orientation" vj_valuelabels { "Rot-X" "Rot-Y" "Rot-Z" } "The Euler XYZ orientation of the dummy"
  end
