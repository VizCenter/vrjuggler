File Connection Chunk

File Connection Chunk token: ``FileConnect''
Defines an output (or, in the future, input) file for the VR Juggler Environment Manager. Its most common use is to define a file which can be used as a destination for performance measurement data.
File Name Gives the pathname for this file. Exactly what constitutes a valid name depends on the underlying file system. On Unix systems, this might be something like "output_data" or "~cjust/tmp/perfdata.txt".
Enabled This property is currently unused.
File Mode For any EM Connection you are creating for a config file, this should be "Output". Internally, Juggler has some ability to read instructions from files or sockets. In the future, this may also allow the use of input instruction files.

File Connection chunk definition

chunk FileConnect "EM Connection" "A file for outputing Environment Manager data"
