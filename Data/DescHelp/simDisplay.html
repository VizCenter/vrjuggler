Simulator Display Chunk

Simulator Display Chunk token: ``simDisplay''
A simulator-mode display window.
Origin Property token: ``origin''
The position on screen, in pixels, of the window's lower-left corner. The upper left corner of the screen is (0,0).
Size Property token: ``size''
The size of window to open, in pixels.
Pipe Property token: ``pipe''
The display pipe to open on. See the Display System chunk for more info.
View Property token: ``view''
The display can be set to draw the viewer's left or right eye view, or it can be instructed to open a stereo view. If the system can't open a stereo display, it falls back to mono.
Border Property token: ``border''
Should the window have a border or not? You probably want this to be true. Making a window borderless on X depends on having a window manager that is mostly compatible with the Motif Window Manager, mwm.
Camera Position Property token: ``cameraPos''
A Position Proxy or Alias to use as a ``camera position'' for this display.
User Property token: ``user''
A Juggler User that controls this display.
Wand Position Property token: ``wandPos''
A Position Proxy or Alias for a user's wand position. Juggler will provide a visual representation of the wand in the simulator window.
Active Property token: ``active''
A display that isn't active is part of the system, but no window will be opened for it and no rendering will occur. This is more useful when testing Surface Display configurations with a Simulator Display.
Draw Projections Property token: ``drawProjections''
If this is true, the Sim Window will display the viewing frustrums used for calculating each surface display.
Surface Color Property token: ``surfaceColor''
Simulator windows can display the positions of all configured Surface Displays. This is useful for testing the needs or usefulness of navigation, etc. This option defines the RGB color that will be used to draw the surfaces.

Simulator Display chunk definition

chunk simDisplay "Simulator Display" "This is a surface that images are projected on."
  Name String 1 "Name" "Unique name of an instance of this chunk type"
  origin Int 2 "origin" vj_valuelabels { "X" "Y" } "Window origin (upper left is 0,0)"
  size Int 2 "size" vj_valuelabels { "Width" "Height" } "Window size"
  pipe Int 1 "pipe" "Graphics pipe"
  view Int 1 "view" vj_enumeration { "left_eye=1" "right_eye=2" "stereo=3" } ""
  border Bool 1 "border" ""
  cameraPos Chunk 1 "cameraPos" vj_enumeration { "PosProxy" "proxyAlias" } 
          "This is the pos proxy that controls the sim view"
  user Chunk 1 "user" vj_enumeration { "JugglerUser" } 
          "This is the user that the window is drawing for."
  wandPos Chunk 1 "wandPos" vj_enumeration { "proxyAlias" "PosProxy" } 
          "The wand pos proxy to use in the simulator."
  active Bool 1 "active" "Is the window active?"
  drawProjections Bool 1 "drawProjections" 
          "Should projections be drawn in the simulator?"
  surfaceColor Float 3 "surfaceColor" vj_valuelabels { "red" "green" "blue" } 
          "What color should the surfaces be?"
  end
