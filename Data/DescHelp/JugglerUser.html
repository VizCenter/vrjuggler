Juggler User Chunk

Juggler User Chunk token: ``JugglerUser''
A Juggler user for use with multi-user applications.
Head Position Property token: ``headPos''
The device from which the head position comes. This must point to a proxyAlias chunk or to a PosProxy chunk.

Juggler User chunk definition

chunk JugglerUser "JugglerUser" "This is a user in Juggler.  This is used for multi-user applications"
  Name String 1 "Name" "Unique name of an instance of this chunk type"
  headPos Chunk 1 "headPos" vj_enumeration { "proxyAlias" "PosProxy" } "This is the source of head position information."
  end
