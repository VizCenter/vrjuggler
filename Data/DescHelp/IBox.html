IBox Chunk

IBox Chunk token: ``IBox''
Configuration Information for an IBox.
Port Property token: ``port''
The string name of the serial port this device is attached to. On Unix systems, a typical value would be something like ``/dev/ttyd3''.
Portnum Property token: ``portnum''
Basically, this is the integer part of the port string above, such as the 3 in ``/dev/ttyd3''.

Yes, this is redundant. Due to peculiarities of the provided IBox driver, the portnum must be correctly filled in. The next version of Juggler will correct this and rely solely on the Port string property.

Baud Rate Property token: ``baud''
The speed of the serial port connection, in bps. Typical values might include 19200 and 38400.

IBox chunk definition

chunk IBox "IBox" "Configuration for the IBox"
  Name String 1 "Name" "Unique name of an instance of this chunk type"
  port String 1 "Port" "Serial port this device is connected to"
  portnum Int 1 "portnum" ""
  baud Int 1 "Baud Rate" "Serial port speed"
  end
