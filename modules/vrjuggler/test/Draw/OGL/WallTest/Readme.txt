WallTest draws a test pattern in 3D on all the walls in the application
It also draws a cube on the left eye, and a tetrahedron on the right eye

If ran in sim mode (If the number of SurfaceDisplays == 0) then it displays a
c6.

Controls:
When Button 0 is held down, the light is attached to the wand
Button 1 toggles lighting on/off (Lighting is off initially)
Button 2 toggles a line of cubes through the screen on/off
	(Cubes are 1/2 foot across, for measuring)
Button 3 toggles the cube/tetrahedron display on/off
