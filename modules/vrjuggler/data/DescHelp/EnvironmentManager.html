Environment Manager Chunk

The Environment Manager is the part of a VR Juggler application which connects to outside control and data-gathering programs, most notably the VjControl configuration editor.

Environment Manager Chunk token: ``EnvironmentManager''
Configures the network settings for the Environment Manager and controls which features are activated.
Port Tells VR Juggler what network port to listen for connections on. A reasonable default value is port 4450.
Accept Connections Tells VR Juggler whether or not it should open the network port and accept connections. If this is set to false, VjControl will be unable to connect to the Juggler application.
Performance Target This field tells Juggler where to send the performance data it gathers. The result is either a File Connection or a Network Connection. Note that currently the Network Connection is not fully implemented.

Environment Manager chunk definition:

chunk EnvironmentManager "Environment Manager" ""
  Name String 1 "Name" ""
  Port Int 1 "Port" ""
  AcceptConnections Bool 1 "Accept Connections" ""
  PerformanceTarget Chunk 1 "Performance Target" 
       vj_enumeration { "FileConnect" "NetworkConnect" } ""
  end
