API Performer Chunk

API Performer Chunk token: ``apiPerformer''
Provides configuration information used only by Performer-based applications.
Sim Head Model Property token: ``simHeadModel''
The full path to the model used for the head. This is a string value, and as the name of a file, it depends upon the underlying file system for its syntax. On UNIX, file names are case sensitive, and directories are separated by the `/' character. On Win32-based systems, file names are not case sensitive, and directories are separated by the `\' character.
Sim Wand Model Property token: ``simWandModel''
The full path to the model used as the wand. This is a string value, and as the name of a file, it has the same restrictions as Sim Head Model.

chunk apiPerformer "apiPerformer" "This is information for configuring the Performer API.  It only pertains to Performer."
