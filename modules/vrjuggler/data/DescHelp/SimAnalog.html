Analog Simulator Chunk

Sim Analog Chunk token: ``SimAnalog''
Defines a simulated analog device. It uses a user-specified keyboard proxy to accomplish this simulation.
Keyboard Proxy Property token: ``keyboardProxy''
Sets the proxy pointing to the keyboard to use for the simulation. The proxy must point to a KeyboardProxy chunk. More chunk types can be added through the chunk description file (chunksDesc).
Up KeyPresses Defines a keypresses to increase the values of the analog inputs. The number of Keypress lines determines the number of analog inputs supplied by this simulator. The user can add and remove keypress lines. Each keypress consists of two parts: a modifier and a key.

Modifier Keys
Lists the modifier keys that may be used together with normal keyboard keys and mouse buttons when increasing the value returned by the analog device. The modifier keys must be chosen from a predefined set of keys that are treated as modifiers. The current modifier keys are:

  • NONE (no modifier)
  • SHIFT (the <SHIFT> key)
  • CTRL (the <CTRL> or <CONTROL> key)
  • ALT (the <ALT> or <META> key)

Keys
The list of keyboard keys, mouse movement directions and mouse buttons that can be used for increasing the value returned by the simulated analog device. All standard keys (except the F1-F12 keys), modifier keys (listed above), cursor movement keys, mouse movement directions and mouse buttons (for a three-button mouse) are available for use. Each key can be assigned to a particular aspect of the simulation, and modifiers can be used to allow the same keys (say those on the number pad) to be used for different aspects of simulation.

Down KeyPresses Defines keypresses to reduce the values of the analog inputs. This works the same as the "Up" keypresses defined above.
Analog Step Property token: ``anastep''
The integer amount that the analog data changes for each key press.

Sim Analog chunk definition

chunk SimAnalog "Analog Simulator" "Simulated analog device.  Gets input from keyboard proxy."
  Name String 1 "Name" "Unique name of an instance of this chunk type"
  keyboardProxy Chunk 1 "Keyboard Proxy" vj_enumeration { "KeyboardProxy" } 
         "The keyboard to use for event input."
  keyPairsInc EmbeddedChunk -1 "Increment Keypress" vj_enumeration { "KeyModPair" } 
         "Keys for moving value up"
  keyPairsDec EmbeddedChunk -1 "Decrement Keypress" vj_enumeration { "KeyModPair" }
         "Keys for moving value down"
  anastep Int 1 "Delta" "The amount that the analog data changes per keypress"
  end
