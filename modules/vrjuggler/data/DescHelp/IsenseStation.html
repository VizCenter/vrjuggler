Intersense Tracker Station Chunk

NOTE: ConfigChunks of this type occur only inside Isense ConfigChunks. You should not create Chunks of this type manually.

Intersense Tracker Station Chunk token: ``IsenseStation''
Configuration for Intersense IS-9000 tracker stations.
Enabled Property token: ``enabled''
Whether or note this station of the tracking system is enabled.
Station Number Property token: ``stationIndex''
This index is used by the tracking system itself, to determine which physical station is referred to. It has no relation to VR Juggler's mapping of inputs to units of the Isense device.
Use Digital Property token: ``useDigital''
Whether or not this station provides digital inputs. Setting this to true for a station without digital inputs may cause the tracking system to malfunction or crash.
Use Analog Property token: ``useAnalog''
Whether or not this station provides analog inputs. Setting this to true for a station without analog inputs may cause the tracking system to malfunction or crash.
First Digital Index Property token: ``digitalFirst''
The first digital index for the Isense device that will contain data from this station. This value is used for mapping inputs from stations to digital units provided by the Isense driver to VR Juggler digital proxies.
Digital Count Property token: ``digitalNum''
Number of digital inputs provided by this station.
First Analog Index Property token: ``analogFirst''
The first analog unit for the Isense device that will contain data from this station.
Analog Count Property token: ``digitalNum''
Number of analog inputs provided by this station.
