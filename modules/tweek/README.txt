1. Getting started
==============================================================================

    A. Supported platforms

        Tweek is officially supported on the following platforms:

            IRIX 6.5
            RedHat Linux 7.1 and newer

    B. Minimum requirements (by operating system)

        * All
            GNU make 3.78 or newer
            GNU Autoconf 2.13
            GNU Automake 1.4

        * IRIX
            MIPSpro Compilers 7.3.1.1m or newer

        * RedHat Linux 7.1
            GCC 2.96.69
